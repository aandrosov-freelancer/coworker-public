## 📖 О проекте
Проект представляет собой MVP информационной системы для цифровой трансформации процесса поиска и найма специалистов на фриланс-платформах. 
Внедрение ИИ-алгоритма семантического анализа и ранжирования на базе open-source языковых моделей (LLM) позволяет исключить ручной этап отсева кандидатов, сократить время первичного подбора не менее чем на 40% и обеспечить прозрачность принятия решений.

## ⚠️ Решаемые проблемы
* **Высокий уровень нерелевантных откликов:** Традиционный поиск по тегам выдает до 70% нерелевантного «шума».
* **Затраты времени:** Заказчики тратят 15–30 минут на ручной анализ одного кандидата (суммарно 30-50 минут на задачу).
* **«Холодный старт»:** Новые фрилансеры без отзывов не попадают в выдачу, даже если обладают нужными навыками.

## 🚀 Ключевые возможности
1. **Управление профилями:** Разделение на заказчиков и исполнителей.
2. **Биржа задач:** Публикация проектов, поиск, фильтрация и отклики.
3. **Встроенный чат:** Бесшовная коммуникация внутри платформы.
4. **ИИ-ранжирование (Zero-shot LLM):** Система формирует промпт с описанием задачи и профилями откликнувшихся, а LLM возвращает отсортированный список наиболее подходящих кандидатов с текстовым обоснованием выбора.

## 🛠 Технологический стек
* **Клиентская часть:** Flutter (единая кодовая база для Web, iOS, Android).
* **Серверная часть:** Supabase (Backend-as-a-Service), PostgreSQL, Edge Functions на TypeScript (Deno).
* **Модуль ИИ:** Интеграция с DeepSeek API.

## 📈 Результаты и экономический эффект
* Время анализа откликов сокращено в 3–5 раз (до 5–10 минут на задачу).
* Снижение рутинной работы для нанимателя.
* Решение проблемы холодного старта благодаря объективному семантическому анализу компетенций исполнителя.
